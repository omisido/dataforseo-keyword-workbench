# Design System Rules

1. Use CSS custom properties for colours, spacing, radii and shadows.
2. Use the defined spacing scale.
3. Every interactive control has a visible focus style.
4. Every form control has an associated label.
5. Use semantic HTML and native controls.
6. Avoid nested cards and decorative gradients.
7. Keep operational screens compact, calm and easy to scan.
8. Do not place credentials or secrets in client-side code.
